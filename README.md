# w-ui

## Install
```
npm install whg-ui
```

### Quick Start
```js
import WhgUI from 'whg-ui'
import 'whg-ui/lib/w-ui.css'
Vue.use(WhgUI)


<template>
    <w-button type='primary'>wui按钮</w-button>
</template>
```

### Keywords
```
whg-ui ui vue components
```
